### Run
```
npm start
```

### Supported operators
- `+` `-` `*` `/` `^` `()`

### Sample I/O
```
Input : 100 - 50 + 10 / 2 *5
Output: 75

Input : 100-(50+10)/2*5
Output: -50

Input : 2^3^4
Output: 2.4178516392292583e+24

Input : (2^3)^4
Output: 4096
```


### Known issues
- Can not handle negative number
- No unary operator supported
- `{}`, `[]` are not supported
